Pada modul ini, kalian akan menggunakan Logistic Regression.
Logistic Regression adalah model machine learning yang digunakan untuk prediksi biner (boolean), seperti:

Ya / Tidak
0 / 1
True / False

Model ini tidak memprediksi nilai langsung, melainkan peluang (probability) dari suatu kelas, yang kemudian dikonversi menjadi hasil biner menggunakan threshold (biasanya 0.5).

Contoh penggunaan:

Spam / non-spam
True / False pada suatu kondisi
